# C-Compiler

A Tiny C compiler(demo).

## Build and run

```
make
./compiler
```

## Feature

### Data type

- `char`
- `int`
- `double`
- `string`

### Operator

- `+` `-`
- `*` `/` `%`
- `==` `!=` `<` `>` `<=` `>=`
- `&&` `||`

## Commit

- Commit to master branch only if the program can build and run.
