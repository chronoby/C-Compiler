# C-Compiler

A Tiny C compiler(demo).

## Build

```
make
```

## Run
```
./compiler <FILE> [options]
```
use `./compiler --help` or `./compiler -h` for usage.

the `test.sh` file is no longer useful.

## Feature

### Data type

- `char`
- `int`
- `double`
- `string`

### Operator

- `+` `-`
- `*` `/` `%`
- `==` `!=` `<` `>` `<=` `>=`
- `&&` `||`
- `++` `--`


## Commit

- Commit to master branch only if the program can build and run.
